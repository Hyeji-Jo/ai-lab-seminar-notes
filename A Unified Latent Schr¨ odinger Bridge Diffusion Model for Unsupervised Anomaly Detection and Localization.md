## 1. 연구 배경 및 문제 정의

### Unsupervised Anomaly Detection
- 비지도 이상 탐지는 산업 현장에서 제품 결함을 자동 감지하거나, 영상 내 특정 이상 징후를 탐지하기 위한 중요한 과제
- 정상/이상 데이터 쌍의 수집이 어렵기 때문에 **Unsupervised 방식이 주로 활용됨**

### 기존 Reconstruction 기반 접근법의 한계
- 대부분의 기존 방법은 정상 이미지로부터 학습된 재구성 네트워크를 사용
- 이상 이미지에 대해 **복원이 어려운 특징**을 기반으로 이상 판단
- 문제점
  - 도메인 적응이 어려움
  - 특수한 상황에 따라 네트워크 구조를 자주 변경해야함
  - 다양한 데이터 분포에 대한 **유연성 부족**


## 2. 제안 방향: Schrödinger Bridge 기반의 Latent Diffusion

- 기존 DDPM이나 Score-based 모델은 **고정된 노이즈 분포**에서만 학습 → 분포 간 전이 어려움
- 본 논문은 **Schrödinger Bridge(SB)** 개념을 활용하여
  - **분포 간 유연한 매핑**이 가능한 구조 설계
  - 이상을 **강조(soft transform)**하는 방식으로 Anomaly Localization 가능


## 3. Schrödinger Bridge(SB)란?

### 개념 요약
- SB는 **두 확률 분포 간의 최적 확률 경로(flow)**를 찾는 수학적 프레임워크
- 예: 정상 이미지 분포 → 이상 이미지 분포 사이의 **최적 확률 흐름**을 모델링

### 수학적 정의
- 두 분포 P_0, P_1가 주어졌을 때, 이들 사이를 연결하는 모든 확률 경로 중
  - 기준 경로(브라운 운동 등)에 대해 **KL Divergence**가 최소인 경로를 찾음
- 이를 통해 얻는 확률 흐름이 **Schrödinger Bridge**


## 4. Static Schrödinger Bridge Problem

### 목적
- 시간에 따른 경로가 아닌, **초기 분포와 최종 분포 사이의 결합 분포**를 최적화하는 문제

### 수학적 의미
- 목적 함수는 크게 두 항으로 구성
  - **1)** 수송 비용(term 1): 두 분포 사이의 유클리디안 거리 기대값 → 이동 효율성
  - **2)** 엔트로피 항(term 2): 경로 다양성을 보존 → 부드러운 경로 유도

### 결론적으로
- **단일 결정 경로로 수렴되는 것을 방지하고**, 확산적인 매핑을 통해 유연한 분포 전이 가능


## 5. 발표자가 강조한 포인트 요약

- 기존 diffusion 방식보다 **SB는 도메인 적응과 일반화 측면에서 유리**
- Score-based diffusion 대비, **reconstruction 목적이 아닌 변환 기반 표현이 특징**
- 논문에서는 **정상 ↔ 이상 분포 간 학습 가능한 유연한 트랜스포트 매핑을 달성**하는 데 초점을 둠
